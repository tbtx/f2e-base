## 版本格式

主版本号.次版本号.修订号，版本号递增规则如下：

1. 主版本号：当你做了不兼容的API 修改
2. 次版本号：当你做了向下兼容的功能性新增
3. 修订号：当你做了向下兼容的问题修正。
