slide 轮播

参考 ## 调用方法 

    var o = new tbtx.Slide(selector, options);
    or
    var o = $(selector).Slide();

    or 使用data-api
    data-ride="carousel"的元素将自动初始化
    <div id="mySlide3" data-ride="slide" data-interval="3000" data-direction="y" class="slide-wrapper">

    prev, next的元素请指<a href="#mySlide4" data-slide="pause"> 或者指定data-target=""

    切换到指定的item： <li data-target="#mySlide3" data-slide-to="0">1</li>

    通过$(selector).data('tbtx.slide')来获取Slide实例的引用
    切换开始触发slide事件，结束触发slid事件

## html结构 

    div.slide-wrapper
        div.slide
            div.slide-inner
                div.slide-item*n

        ol.indicators
        prev
        next
.slide-wrapper指定position relative方便prev/next 以及indicators绝对定位，类名可以更改 .slide指定可视区域的大小，默认会读取其width和height，在指定itemSize的情况下可以更改 .slide-inner 类名不能更改，用来调整margin的元素 需要指定.slide-item的大小 ## 默认参数 

        defaults: {
            interval: 3000, // 切换间隔
            duration: 600, // 动画时间
            viewSize: 1,        //  在可见区域有几个item
            itemSize: 0, // 每个item的size，当x方向滚动时指的是width, y-height

            itemSelector: '.slide-item',    
            indicatorSelector: '.indicators li',
            activeClass: 'active',

            // 是否可以沿一个方向无限循环
            isCarousel: true,
            // 移动方向
            // x, y, or none
            direction: 'x'
        }
