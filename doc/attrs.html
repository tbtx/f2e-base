tbtx.Aspect

来自支付宝arale

需要与tbtx.Events tbtx.Class配合使用

属性set/get

例如：


    var Overlay = new Class;
    Overlay.Implements([tbtx.Events, tbtx.Attrs]);

    var o = new Overlay();

    var config = {
        name: 123
    };
    o.initAttrs(config);

    o.on("change:name", function(value, prev, key){ alert(123)});
    o.set("name", 456);

想让某个对象拥有该能力:


    var o = {};
    tbtx.classify(o); // 拥有Implements方法
    o.Implements([tbtx.Events, tbtx.Attrs])

    o.initAttrs(config);
