tbtx.Aspect

需要与tbtx.Events tbtx.Class配合使用

在调用某个方法前后注入方法

例如：/p> 


    var Overlay = new Class;
    Overlay.Implements([tbtx.Events, tbtx.Aspect]);

    var o = new Overlay();
    // 在Overlay的show方法之前调用该方法，如果return false则show方法不会调用
    o.before("show", function() {
        console.log('before show');
        return false;
    });

想让某个对象拥有事件能力:


    tbtx.Events.mixTo(o)

优先向o.prototype复制， 其次是o

默认 tbtx.Events.mixTo(tbtx)

也就是tbtx下面有了on/off/trigger三个方法
